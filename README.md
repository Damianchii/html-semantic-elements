### Semantyczne Elementy - sama nazwa znacznika zdradza co może znajdzować się wewnątrz tego znacznika

## ➡️ `<header>` - nagłówek
```bash
  <header>
        Nagłowek storony - zazwyczaj wewnątrz znajduję się nawigacja
  </header>
```
